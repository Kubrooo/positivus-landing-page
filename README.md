# Positivus Landing Page

A clean, modern, responsive landing page template for digital marketing agencies.  
Built using HTML, CSS, and JavaScript.

Positivus adalah template landing page yang dirancang untuk agensi digital marketing. Desainnya minimalis dan modern, dengan penggunaan warna hitam, putih, dan aksen hijau cerah. Template ini mencakup:

- Header & hero section
